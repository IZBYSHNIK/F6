# F6 — Электронный журнал успеваемости и посещаемости

[![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)
[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![PySide6](https://img.shields.io/badge/PySide6-6.x-brightgreen)](https://doc.qt.io/qtforpython-6/)
[![Version](https://img.shields.io/badge/version-1.2.0-orange)](https://github.com/IZBYSHNIK/F6/releases)

**F6** — это десктопное приложение на Python и PySide6 для классных руководителей, старост и заведующих отделениями. Оно позволяет вести учёт прогулов (по уважительным и неуважительным причинам), выставлять оценки, загружать расписание из документов Word, экспортировать данные в Excel, строить графики успеваемости и многое другое.

![Логотип](logo.png)

## 📋 Основные возможности

- **Многопользовательский режим** — каждый пользователь имеет свой профиль, данные хранятся в `~/F6/`.
- **Учёт посещаемости**  
  - Прогулы по уважительной причине (ПОУВ) / неуважительной (НЕУВ).  
  - Автоматический расчёт человеко-часов, качества посещаемости.
- **Учёт успеваемости**  
  - Выставление оценок (2–5) по предметам.  
  - Статистика: количество студентов с 2, с одной 3, с 5, качественная и общая успеваемость.
- **Загрузка расписания**  
  - Импорт из файлов `.doc` / `.docx` (с автоматической конвертацией).  
  - Парсинг количества пар по дням.
- **Гибкое управление рабочими днями**  
  - Редактирование рабочих/нерабочих дней вручную.  
  - Назначение праздничных дней (двойным кликом).
- **Архивация**  
  - Завершение месяца → таблицы сохраняются в архив.  
  - Просмотр архивных месяцев, восстановление.
- **Экспорт в Excel** (`.xlsx`) — посещаемость, оценки, архивы.
- **Графики и статистика**  
  - Круговая диаграмма доли прогулов по студентам.  
  - Линейный график динамики прогулов за несколько месяцев.
- **Достижения (ачивки)**  
  - Скрытые награды за различные действия (полный набор студентов, завершение месяца, смена иконки, исследователь и др.).
- **Локализация** — русский, английский, китайский (переключается по клику на иконку).
- **Обновление через GitHub** — проверка последней версии при запуске.

## 🖥️ Технологии

- **Язык**: Python 3.8+
- **GUI**: PySide6 (Qt for Python)
- **Графики**: Matplotlib
- **Обработка документов**: python-docx, doc2docx
- **Сборка и зависимости**: pip / requirements.txt
- **Хранение данных**: JSON (пользователи, настройки, праздничные дни), JSON/бинарные файлы (студенты, оценки, прогулы)
- **Лицензия**: Apache 2.0

## 📦 Установка и запуск

### Требования

- Python 3.8 или новее
- pip (менеджер пакетов)
- Рекомендуется виртуальное окружение

### Установка зависимостей

```bash
# Клонирование репозитория
git clone https://github.com/IZBYSHNIK/F6.git
cd F6
pip install -r requirements.txt
python main.py
```
> При первом запуске будет создана папка ~/F6/ (на Linux/macOS) или %USERPROFILE%\F6\ (на Windows), где будут храниться данные пользователей, архивы и настройки.

## 🎮 Использование

### 1. Авторизация / Регистрация

- Выберите пользователя из списка или создайте нового через «Регистрация».
- При регистрации указываются: логин, пароль, ФИО, группа, специальность, ФИО классного руководителя.

### 2. Главные вкладки

| Вкладка       | Назначение                                                                                   |
|---------------|----------------------------------------------------------------------------------------------|
| **Прогулы**   | Таблица дней месяца / студентов. Зелёные ячейки — ПОУВ, оранжевые — НЕУВ. Двойной клик по пустой ячейке → добавить прогул, повторный — удалить. |
| **Оценки**    | Аналогичная таблица для выставления отметок (2–5).                                           |
| **Студенты**  | Просмотр, редактирование, удаление студентов.                                                |
| **Архив**     | Список завершённых месяцев, просмотр, активация, удаление.                                   |
| **Настройки** | Изменение рабочего месяца, сброс праздников, очистка таблиц, отключение вкладок, смена языка. |
| **Статистика**| Круговая диаграмма прогулов за текущий месяц или линейный график динамики за несколько месяцев. |
| **Профиль**   | Редактирование личных данных, смена пароля, удаление аккаунта, уровень защиты данных (минимальный / средний / высокий), коллекция достижений. |

### 3. Работа с расписанием

- На вкладке «Прогулы» нажмите на кнопку с иконкой загрузки (📅).
- Укажите папку с файлами `.doc` или `.docx`, содержащими расписание (по дням).
- Программа распарсит количество пар и автоматически заполнит часы.

### 4. Сохранение и экспорт

- **Зелёная дискета** — сохранить изменения.
- **Красная кнопка «Game Over»** — завершить текущий месяц (создать архив).
- **Кнопка Excel** — экспорт текущей таблицы в `.xlsx` для печати.

### 5. Достижения

Открываются при выполнении особых условий. Примеры:

| Название                      | Условие получения                          |
|-------------------------------|---------------------------------------------|
| Full_stack (Полный набор)     | Заполнить в таблицу 25 студентов           |
| Игра окончена                 | Завершить первый раз рабочий месяц         |
| Трансформация                 | Сменить иконку профиля                      |
| Шерлок?                       | Ввести специальность: «Великий сыщик»       |
| Исследователь                 | Посетить все вкладки в программе            |

Все достижения отображаются на вкладке «Профиль».

## 📁 Структура хранения данных
```text
~/F6/
├── users.json                 # Список пользователей (ID, имя, общие настройки)
├── <user_id>/                 # Папка каждого пользователя
│   ├── user.json              # Логин, хэш пароля, параметры, достижения, уровень защиты
│   ├── students.json          # Данные текущего месяца: студенты, прогулы, оценки, дни
│   ├── happy_days.json        # Глобальные праздничные дни (на все годы)
│   └── archive/               # Архивные копии завершённых месяцев (JSON-файлы)
```
